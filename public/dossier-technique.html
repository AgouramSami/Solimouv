Choix stratégique

PWA plutôt qu'application native

Pour un festival annuel avec une audience large et des contraintes de budget nulles, une PWA est la solution optimale.

✓ PWA — Notre choix

  • Installable sur iOS + Android depuis le navigateur
  • Zéro friction : pas d'App Store, pas de compte
  • Un seul codebase, déploiement instantané
  • Mise à jour transparente (pas de validation store)
  • Fonctionne hors-ligne (Workbox cache stratégies)
  • SEO natif, indexable par Google
  • Coût infrastructure : 0€ sur Vercel (hobby tier)

Application native iOS/Android

  • 2 codebases distincts à maintenir
  • Délais validation App Store (2-7 jours)
  • Compte développeur Apple 99$/an
  • Installation obligatoire avant usage
  • Compétences iOS/Android requises
  • Inadapté pour un événement annuel
Architecture

Stack technique

Stack moderne orientée productivité : Next.js full-stack pour réduire la surface de code, Supabase pour éliminer le backend custom, Make.com pour les automatisations sans serveur.

Frontend

Next.js 15 — App Router + React Server Components

TypeScript · Tailwind CSS · Cabinet Grotesk + Author (Fontshare)

→ Full-stack en un seul repo, SSR/SSG natif, optimisation images automatique, routing file-based. Adoption massive, documentation exhaustive.
PWA

@ducanh2912/next-pwa + Workbox

Web App Manifest · Service Worker · Cache stratégies (NetworkFirst, StaleWhileRevalidate)

→ Génération automatique du service worker, cache offline, installabilité iOS/Android sans configuration complexe.
Backend / BaaS

Supabase — PostgreSQL + Auth + Storage + Realtime

RLS (Row Level Security) · Edge Functions · API REST auto-générée

→ Remplace un backend Node/Express entier : authentification, base de données, stockage fichiers, temps réel — zéro infrastructure à gérer. Tier gratuit suffisant pour le MVP.
Automation

Make.com (ex-Integromat)

Webhooks → Google Sheets CRM · Emails Gmail · Notifications push

→ Connecte la PWA à l'écosystème existant de l'association (Google Sheets, Gmail) sans développement backend. Scénarios visuels maintenables par l'équipe non-technique.
Hébergement

Vercel — Hosting + CI/CD

Deploy automatique sur push Git · Preview URLs par branche · Edge Network mondial

→ Intégration native Next.js (même éditeur), déploiement en 30 secondes, CDN mondial, HTTPS automatique. Tier Hobby gratuit couvre largement un festival.
